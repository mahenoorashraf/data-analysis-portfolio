# Data Analysis Portfolio

This repository contains a **portfolio of 5 complete data analysis projects** covering multiple domains including **retail, education, weather, healthcare, and finance**. Each project includes **data cleaning, analysis, visualizations, and actionable business insights**.

---

## 📁 Repository Structure
datasets/ → Raw datasets used in projects
notebooks/ → Jupyter notebooks for each project
visualizations/ → Generated charts and plots
reports/ → PDF reports for each project
requirements.txt → Python dependencies for running notebooks
README.md → Project overview and documentation

---

## 📊 Projects Overview

| Project | Description | Key Visualizations | Insights & Recommendations |
|---------|-------------|-----------------|----------------------------|
| **Project 1: Supermarket Sales Analysis** | Analyze daily sales patterns, best-selling products, and customer behavior | Correlation Matrix, Customer Rating Distribution, Customer Gender Distribution, Sales by City | Increase electronics stock by 20%, Launch targeted weekend promotions |
| **Project 2: Student Performance Analysis** | Evaluate student grades, pass/fail rates, study time impact, and gender performance | Average Student Grades, Pass vs Fail Distribution, Average Grades by Gender, Impact of Study Time | Personalized support for low-performing students, Encourage effective study hours |
| **Project 3: Weather Data Analysis** | Analyze temperature, wind speed, pressure, humidity, and seasonal patterns | Temperature Trend Over Time, Average Temperature by Month, Wind Speed Distribution, Pressure Distribution, Humidity Distribution, Weather Correlation | Identify seasonal trends, Extreme weather alert planning |
| **Project 4: Stock Market Analysis** | Analyze cryptocurrency, gold, and stock price trends | Bitcoin Price Trend, Gold Price Trend, Bitcoin vs Ethereum, Market Correlation Heatmap, Tech Stock Price Comparison, Gold vs Bitcoin Price Trend | Investment insights, Risk assessment, Market trend evaluation |
**Project 5: COVID-19 Trend Analysis** | Daily cases, total cases, deaths, and correlations in India | Daily COVID Cases in India, Total COVID Cases in India, Total Deaths, COVID Data Correlation | Track spread trends, Evaluate effectiveness of interventions |
